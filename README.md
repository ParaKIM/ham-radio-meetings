# 아마추어무선 모임 참석자 관리

React 기반의 localStorage 참석자 관리 웹앱입니다.

## 기능

- 모임별 참석자 명단 관리
- 호출부호 기준 참석자 관리 및 중복 호출부호 감지
- 회비와 찬조금 입력 및 만원 단위 계산
- 장소명, 장소 주소, 모임 날짜, 참석 인원, 총액 표시
- 다음 모임 생성 시 기존 참석자 명단 불러오기
- CSV 저장

## 실행

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```

## 배포

빌드 결과는 `gh-pages` 브랜치에 배포되며 GitHub Pages에서 정적 사이트로 제공됩니다.
